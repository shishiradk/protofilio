HOW TO OPEN ADMIN PANEL
=======================

Since you requested to remove the Admin link from the main website, you must open the admin panel manually.

Option 1: From File Explorer
----------------------------
1. Go to this folder on your computer.
2. Find the file named "admin.html".
3. Double-click it to open it in your browser.

Option 2: From Browser URL
--------------------------
1. if your website URL looks like:
   file:///C:/.../index.html
   
2. Change the end to:
   file:///C:/.../admin.html

TROUBLESHOOTING
---------------
- If the page is blank, ensure "data.js" and "admin.js" vary in the same folder.
- If changes don't save, make sure to click "Save & Download" and replace the "data.js" file.
